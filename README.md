# five-channel-responder
A five-channel digital responder (quiz buzzer) circuit design using TTL logic ICs (74LS series). Features priority encoding, latching, countdown timer (16s), and LED indicators. Course project for Digital Electronics.基于TTL逻辑芯片（74LS系列）的五路数字抢答器电路设计。具有优先编码、信号锁存、16秒倒计时和LED指示功能。数字电子技术课程设计项目。
